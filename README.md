<h1>Wrisp</h1>
<p>
  A note-taking app built with <b>React</b> and <b>TypeScript</b> in a <b>Vite</b> environment. 
  This is based on a project demonstrated on the YouTube video 
  <a href="https://youtu.be/j898RGRw0b4" target="_blank" rel="noopener noreferrer">
    The Perfect Advanced React/TypeScript Project - Markdown Supported Note Taking With Categories
  </a> by <b>Web Dev Simplified</b>.
</p>

<hr>

<h2>Features</h2>
<ul>
  <li><b>Note management</b> - Create, edit, and delete notes seamlessly.</li>
  <li><b>Tag system</b> - Add and manage multiple tags per note.</li>
  <li><b>Search-based filtering</b> - Instantly filter notes with search keywords.</li>
  <li><b>Local storage utilization</b> - Uses browser storage to save your notes.</li>
  <li><b>Responsive UI</b> - Works smoothly across desktop and mobile device layouts.</li>
</ul>

<h2>Installation</h2>

<h3>Prerequisites</h3>
<ul>
  <li>Ensure you have <b>Node.js</b> installed</li>
</ul>

<p>
  Download Node.js (Windows Installer):

<h3>Getting Started</h3>

<pre><code># Clone this repository
git clone &lt;your-repo-url&gt;
  
# Navigate into the project directory
cd wrisp
  
# Install dependencies
npm install
  
# Start the development server
npm run dev</code></pre>
